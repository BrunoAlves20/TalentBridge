# Paleta de Cores do Projeto (Talent Bridge)

O projeto utiliza as cores padrão do **Tailwind CSS**, organizadas nas seguintes categorias de uso:

## 🔵 Cores Principais (Azul)
Utilizadas principalmente para a marca, botões de ação primária, destaques e gradientes.
- **Backgrounds e Gradientes**: `bg-blue-50`, `bg-blue-100`, `bg-blue-600` (Primária), `bg-blue-700` (Hover), `from-blue-50`, `from-blue-600`, `to-blue-700`
- **Textos e Ícones**: `text-blue-100`, `text-blue-400`, `text-blue-500`, `text-blue-600`, `text-blue-700`
- **Bordas**: `border-blue-100`, `border-blue-200`, `border-blue-600`

## ⚪ Neutras (Escala de Cinza e Branco)
Utilizadas para estruturar o layout, textos longos, fundos secundários e o rodapé.
- **Branco**: `bg-white`, `text-white`, `border-white`
- **Backgrounds Claros**: `bg-gray-50`, `bg-gray-200`
- **Textos Secundários**: `text-gray-300`, `text-gray-400`, `text-gray-500`, `text-gray-600`, `text-gray-700`
- **Elementos Escuros (Textos Principais e Footer)**: `text-gray-800`, `text-gray-900`, `bg-gray-900`, `border-gray-800`

## 🟢 Cores de Sucesso e Ação Secundária (Verde)
Utilizadas para feedback positivo (como o Coach de IA), badges e destaques de candidatos preparados.
- **Backgrounds**: `bg-green-100`, `bg-green-400`, `bg-green-500`, `bg-green-600`
- **Textos**: `text-green-500`, `text-green-600`, `text-green-700`

## 🔴 Cores de Alerta (Vermelho)
Utilizadas para interações que requerem atenção, como o botão de gravação do microfone (pulsante).
- **Backgrounds**: `bg-red-400`, `bg-red-500`

## 🟡 Cores de Destaque / Conquista (Amarelo)
Utilizadas para destacar os melhores candidatos ou conquistas (como o ícone de prêmio).
- **Backgrounds**: `bg-yellow-400`

## 🟣 Cores de Acento (Índigo)
Utilizadas para complementar os gradientes junto com o azul.
- **Gradientes**: `to-indigo-50`
